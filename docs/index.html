the pipeline

Six layers, each with one job

layer 1

Capture

Screen Studio for polished demos, raw camera for vlogs, desktop capture via videodb. Collect everything — cutting comes later.

layer 2

Organize

Claude transcribes, labels topics, finds dead sections and repeated takes, and produces an edit decision list with timestamps.

layer 3

Cut

FFmpeg executes the EDL — lossless segment cuts, concats, reframes for YouTube, TikTok, and Instagram aspect ratios.

layer 4

Compose

Remotion for programmatic titles, intros, overlays, and subtitles as React components — versioned like code, not clicked together.

layer 5

Augment

ElevenLabs voiceover where the take failed, fal.ai for b-roll gaps and image assets. Fill holes, don't fabricate the film.

layer 6

Polish

Final pass in Descript or CapCut — the human taste layer. Music, pacing, the last 10% that makes it watchable.
